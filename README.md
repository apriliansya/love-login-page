# Love Login Page

Simple romantic-themed login page with a glassmorphism UI, image background, and video background.

Halaman login sederhana bertema romantis dengan tampilan glassmorphism, latar gambar, dan video background.

## English

### Preview

![Sample Preview](sample.png)

<video src="sample.mov" controls width="100%"></video>

[Open video preview](sample.mp4)

This project includes:
- Minimal login form (username and password)
- Transparent blur login card (glassmorphism effect)
- Combined image and video background (`sakura.mp4`)
- Mobile responsive layout

### Tech Stack

- HTML5
- CSS3

### Project Structure

```bash
Love Login Page/
├── index.html
├── style.css
├── background.jpeg
├── sakura.mp4
├── sample.png
├── sample.mov
└── README.md
```

### How to Run

1. Clone this repository:

```bash
git clone https://github.com/apriliansya/love-login-page.git
```

2. Go to the project folder:

```bash
cd love-login-page
```

3. Open `index.html` in your browser (double-click or use Live Server).

### Quick Customization

- Change page title in `index.html`
- Update button colors in `style.css` (`button` and `button:hover`)
- Replace the background image by changing `background.jpeg`
- Replace the video background and update the source for `sakura.mp4`

### Notes

- The form is currently frontend-only (not connected to backend auth).
- For production use, add input validation and authentication integration.
